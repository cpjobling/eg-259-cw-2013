    js/data/data.js

    This pattern, from Adi Osmani's forthcoming book Backbone.js Fundamentals (available as a free e-book at ensures that a global object called app is available to our code. If app is not already defined, a new empty object called app will be created. If all our objects adopt this pattern and then attach themselves to app, we will have a namespaced object that can be shared as a global context for our application.

    app = app || {}
